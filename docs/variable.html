Variable Font support
Variable Font support for any number of defined axes

Fontsampler supports any number of (manually defined) axes and arbitrary min, max, init and step values for those. The above example shows only a few of Amstelvar’s axes.

Variable Font instances and axes will be ignored if only a woff file (no woff2) are supplied or the browser does not support woff2.
